# Frontend Job Listing UI - Cygnus Assignment

## Objective
A responsive Job Board Interface built using React and Tailwind CSS.

## Features
- Display Job Cards
- Location Filter (Remote / On-site)
- Job Type Filter (Internship / Full-time)
- Real-time Search
- Responsive Design
- Alphabetical Sorting
- Highlight searched keyword

## Tech Stack
- React (Vite)
- Tailwind CSS
- JavaScript (ES6)

## Installation

```bash
npm install
npm run dev
