# Job Listing App

A modern and responsive Job Board application built using **React and Vite**.  
This application allows users to search and filter job listings dynamically with real-time updates.

---

## Live Demo

Deployed Application:  
https://job-listing-app-orpin.vercel.app/
---

## Features

-  Search jobs by title
-  Filter jobs by category
- Real-time dynamic filtering
- Fully responsive design
- Reusable component-based architecture
- Fast development and build with Vite

---

## Tech Stack

- **Frontend:** React
- **Build Tool:** Vite
- **Language:** JavaScript (ES6+)
- **Styling:** CSS
- **Version Control:** Git & GitHub
- **Deployment:** Vercel / Netlify

---

## Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/SahithiGurram10/job-listing-app.git
```

### 2️. Navigate to the project directory

```
cd job-listing-app
```

### 3️. Install dependencies 
```
npm install
```
### 4️. Start the development server 

``` 
npm run dev
```

# The application will run at:
```
http://localhost:5173
```

## Assumptions

- Job data is stored locally in a static data file.
- Filtering is performed on the client side.
- The application does not use a backend or database.
- The project focuses on frontend functionality only.
